## What's inside

90 VC firms mapped across SF, Peninsula & Silicon Valley — color-coded by investment stage:

| Stage | Firms |
|---|---|
| 🌱 Talent-First / Pre-Idea | Entrepreneurs First, 1517 Fund, Bloomberg Beta, Human Capital |
| 🚀 Pre-Seed / First Believers | Afore, Conviction, Pear VC, Precursor, SV Angel, Hustle Fund + more |
| 🌿 Seed-First / Early-Stage | Benchmark, Felicis, First Round, Greylock, CRV, Floodgate + more |
| 🏛️ Multi-Stage / Large Platform | a16z, Sequoia, Khosla, Kleiner Perkins, Lightspeed, General Catalyst + more |
| 🎯 Sector-Focused / Thesis-Driven | NFX, Ribbit Capital, Atomic VC, Obvious Ventures, Lux Capital + more |
| ⚡ Broad Early-Stage | Founders Fund, SignalFire, Draper Associates, Renegade Partners |
| 🏗️ Platform / Accelerator | Y Combinator, Plug and Play |

## Features

- Interactive map built on OpenStreetMap + Leaflet
- Click any pin → firm details, portfolio, partners, website
- Filter by stage, search by name/partner/thesis
- Dark theme

## Stack

Plain HTML + JS. No build step. One `index.html`, one `data.js`.
